
# Infinity Shell - TryHackMe Write-Up

This repository contains my walkthrough for the **Infinity Shell** room on TryHackMe.

In this room, I investigated a compromised web application, identified a malicious web shell, traced the attacker’s activity through Apache logs, and recovered the flag by decoding the commands sent to the shell.
